What ARE Prototypes?

-a property that's part of a function.

-an object that contains native/custom properties & methods.

-use prototypes to simulate class inheritance.

JavaScript doesn't have classes

Classical languages: classes can inherit from other classes.

Prototypal languages: objects can inherit from other objects.

Classes in JavaScript are just a shorter way of using prototypes.

From Effective JavaScript

"Classes in JavaScript are essentially the combination of a constructor function...and a prototype object used to share methods between instances of the class..."
